🌍 Global Airbnb Analytics Dashboard

🚀 An end-to-end analytics project using Power BI, SQL, and Excel to uncover pricing trends, demand patterns, and location-based insights.

📊 Project Overview

This project analyzes global Airbnb listing data to generate actionable business insights. The workflow includes data cleaning, SQL-based analysis, and building an interactive Power BI dashboard to understand pricing behavior, customer preferences, and availability trends.

🎯 Objectives
Analyze pricing trends across different locations
Identify high-demand regions
Understand distribution of property types
Study availability and booking patterns
🗄️ SQL Analysis 
Imported dataset into PostgreSQL database
Performed SQL queries to extract insights:
Top locations by revenue
Average price by property type
Listing distribution across cities
Availability trends
📈 Key Insights
Prices vary significantly across cities and neighborhoods
Certain locations show higher demand and listing density
Entire homes dominate premium listings
Availability trends highlight peak booking periods

Power BI – Dashboard & visualization
