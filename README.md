# 🌍 Global Airbnb Dashboard

🚀 An interactive Power BI dashboard analyzing global Airbnb data to uncover pricing trends, demand patterns, and location-based insights.

---

## 📊 Project Overview
This project focuses on analyzing Airbnb listing data across multiple regions to derive meaningful business insights. The dashboard helps in understanding pricing behavior, customer preferences, and availability trends.

---

## 🎯 Objectives
- Analyze pricing trends across different locations  
- Identify high-demand regions  
- Understand distribution of property types  
- Study availability and booking patterns  

---

## 📈 Key Insights
- Prices vary significantly across cities and neighborhoods  
- Certain locations show higher demand and listing density  
- Entire homes dominate in premium segments  
- Availability trends highlight peak booking periods
